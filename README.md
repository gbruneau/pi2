# Pi (π)
Calculate π using Chudnovsky algorithm

## Compile
```bash
make
```
##  Run
```bash
./pi n # where n is the number of decimals

# example with 25 digits
./pi 25
```

##  Output

- `./output/pi-n.txt` contain π with n decimals
- `./output/pi-n.log` is the matching log file
